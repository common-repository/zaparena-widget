== Description ==

Put the zaparena traffic widget to your blog to get more visitors. Sign up at http://www.zaparena.com

== Installation ==

1. Install the wp-zaparena.zip in the 'Plugin' menu
2. Activate the plugin through the 'Plugins' menu
3. Set up the zaparena widget id
4. Done ;-)

== Frequently Asked Questions ==

= Where do I find the widget id? =

Edit your widget on zaparena, on the code page there is the widget id.
